# Captação de Dados - Casa dos Dados
## Objetivo
Desenvolver um script que abra o navegador no link da pesquisa avançada da Casa dos Dados, espera o usuário selecionar os filtros existentes no site, aguarda confirmação, e raspa os dados de cada empresa listada para dentro de uma planilha
csv separada por vírgulas.

## Requisitos para instalação
- (Python)[https://www.python.org/downloads/] para interpretar os arquivos .py
- (Google Chrome)[https://www.google.com/intl/pt-BR/chrome/] para carregar as páginas.
